E-Commerce Application
 
Project Description:
This is a full-stack E-Commerce web application built with a dedicated backend server and a dynamic frontend. 
The application supports user authentication, product browsing, cart management, and an admin dashboard for product control.

Features Implemented:
User Functionality
User Signup, Login & Logout (Authentication)
Browse Products

Category-based Sorting:
Men
Women
Kids

Add to Cart
Remove from Cart
Automatic Total Price Calculation
Latest Collections Section
Related Products Display

Backend Server:
RESTful API built using Node.js & Express
MongoDB database integration
JWT-based authentication
Protected routes for secure operations
Product CRUD operations
Image upload handling

Admin Panel Functionality:
Add New Products
View Product List
Delete Products
Product uploading
All product images are securely stored and managed using Cloudinary for optimized cloud storage and fast delivery.

Deployment:
Backend deployed on Render
Frontend deployed on Vercel
